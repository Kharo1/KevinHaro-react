# KevinHaro-react
KevinHaro.com now built using reactjs


Commands to run application locally:

  npm run react-dev
  
  npm run server-dev
  
  gulp build
 
gulp build must be ran in the /semantic directory.

Local Site:
  http://localhost:8080/
  
  
Built using:
Semantic UI React
reactJS
webpack
babel
